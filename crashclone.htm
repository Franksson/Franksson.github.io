Crash Bandicoot Clone

Previous Next

The Story

I played alot of crash bandicoot when i was young and recreating a type of clone of that game in a modern day engine seems a lot easier than it probably was when it was first created for the playstation.

I began creating some core components, like coins, boxes, simple enemies and camera movement, so i later could could efortlessly create multiple levels with the assets. I had done movement alot of times before, the real challange was creating a smooth moving camera that followed the player seamlessly.

I first tried creating a path but converting the steps of the path from localposition to worldposition proved to complicated so i settled on creating a camera that just moved forward until the player moved through a trigger that lerped the camera to a new position. This technique proved very efficient and useful.


Whats Next

Tweaking the movements a bit and adding a real level with some more advanced obsticles.
